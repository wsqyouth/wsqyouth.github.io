hexo添加阅读全文

发表于 2019-03-06
| 字数统计: 193 字 | 阅读时长 ≈ 1 分钟

这里学习如何添加阅读全文的功能。

实现在主页只展示部分文字，其他文字隐藏起来，通过点击阅读更多来阅读全文。
方法就是写每一篇文章的时候，在必要的地方添加<!-- more -->即可。
例如：

# hexo # 博客
hexo文本居中的引用
hexo常用命令记录
